== Description ==

Simple slider plugin what will give you a chance to setup a slider easily..

it looks a bit different.. 1 image big sized and other images are thumbnails at the right side of the big image.

please check the screenshots to see how would it look..

Demo: <a href="http://dreamwebit.com/gallery-slider/awesome-slider/">awesome slider</a>

Please check out the demo if you have confusion about the look..

== Installation ==



1. Upload `wpbatch-awesome-slider.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will see a new menu in the dashboard "WPBatch Slider"... add some images from there
4. now just add this shortcode anywhere in the page, post or widgets 
<pre>[wpbatch-slider]</pre>
or Place <pre><?php do_shortcode('[wpbatch-gallery]'); ?></pre> in your templates php files

Good Luck
